2. Dati Raccolti

PropHedge Pro e un software HTML standalone. Funziona interamente nel browser locale: nessun dato viene inviato a server di VISIONTRADING. Tutti i calcoli e le configurazioni rimangono esclusivamente sul dispositivo dell'utente.

Dati trattati localmente (nessuna trasmissione a VISIONTRADING):

  • ◆ Parametri sfide prop inseriti dall'utente (dimensione, DD, giorni, lotti)
  • ◆ Preferenze di configurazione (asset, pip, modalita hedge)
  • ◆ Dati di sessione cancellati alla chiusura del browser

Dati trasmessi a terze parti (solo su richiesta dell'utente):

  • ◆ API Token OANDA - trasmesso direttamente a OANDA plc solo se si usa la funzione broker. VISIONTRADING non riceve ne conserva questi dati.
  • ◆ Tasso di cambio EUR/USD - richiesta anonima a cdn.jsdelivr.net. Nessun dato personale trasmesso.
  • ◆ Favicon broker - caricamento tramite Google Favicon Service (richiesta anonima).
